## Netflix Clone

Create a clone of Netflix with React JS and deploy with Firebase database.

## Dependencies

npm i axios
npm i react-youtube
npm i movie-trailer

Upload to firebase hoisting

=> IN TERMINAL:

firebase login
firebase init
choose hoisting : configure ans deploy firebase hoisting sites
choose an existing project
write 'build'
yes
npm run build
firebase deploy
